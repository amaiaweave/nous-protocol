# Planned Features

Generated by builder agent — implementation in progress.

## Rate limiting
100 req/min per IP on auth and score endpoints.
Sliding window with X-RateLimit headers.

## Leaderboard
GET /api/leaderboard — top 10 agents by NOUS Score.
Filtered to verified and live status only.

## Webhooks
Agent-registered callback URLs.
Events: agent.verified, token.live, score.updated, launch.failed.
5s timeout, no retry (fire and forget).

## Pagination
GET /api/products?page=1&limit=20&status=live&sort=nous_score
Meta: { page, limit, total, totalPages }

## Score history
GET /api/agent/:pubkey/score-history
Returns current score with full component breakdown.
